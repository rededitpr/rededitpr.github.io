Instruções

Computador

  1. Na página de redação, pressione F12 ou CTRL + SHIFT + I para abrir as ferramentas de desenvolvedor do navegador.
  2. Na barra de ferramentas que aparecer, selecione a aba Console.
  3. No campo de entrada do console, digite "allow pasting" e pressione Enter.
  4. Em seguida, copie e cole o código gerado anteriormente no campo do console.
  5. Aguarde até que a barra de transcrição seja concluída.

Android

  1. Instale o aplicativo Eruda - Browser Console.
  2. Acesse a página da redação pelo navegador.
  3. Toque no ícone flutuante de engrenagem para abrir o console.
  4. Selecione a aba Console.
  5. No canto inferior, toque no símbolo > para abrir um campo de texto
  6. Cole o código gerado anteriormente neste site no campo de texto.
  7. Toque em Execute para executar o código.
  8. Aguarde até que a barra de transcrição seja concluída.
